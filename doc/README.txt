Install instructions:
	Server: 
		install postgresql server (create the default user with password: root)
		install the application (build it & run it)
		
	Client:
		install the application (build it & run it)
	
Run the application:
	Make sure the server is running (a console app should say waiting for connections)
	run the client on the same computer
	you can already read articles.
	
	if you want to add or modify articles, you have to use one of the two writer accounts:
		name: first_writer
		password: first
		
		or
		
		name: second_writer
		password: second

For use cases see the UseCases folder in doc/
